Please, dont modify script. To add new station, edit radios.xml. Example: <radio name="POLISH SATAN METAL STATION" path="http://195.94.205.211/rm" type="1" />. Types: 0 - GTA radio, 1 - rock, 2 - electro/dubstep, 3 - 60s/70s/80s 4 - default

Commands: /radios, /stations, /station

If you want to use radio without being in car, change settings in meta.xml
